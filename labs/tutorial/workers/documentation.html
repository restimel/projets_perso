Sources et Documentations

Voici quelques liens qui m'ont permis d'écrire ce tutoriel, plus quelques autres qui pourront vous aider ou vous faire découvrir les workers d'un autre point de vue.

Spécifications

  • whatwg - Web-workers (en) : La description du fonctionnement des workers selon le groupe WHATWG.
  • whatwg - Communications (en) : La description du fonctionnement des envois de messages selon le groupe WHATWG.
  • W3C - Web Workers (en) : La description du fonctionnement des workers selon le W3C.

Documentations

  • MDN - Web-workers (en) : Documentation de l'utilisation des Workers pour Firefox (donc les shared-workers ne sont pas décrit)
  • MSDN - Web Workers (en) : Documentation de l'utilisation des Workers pour IE (donc les shared-workers ne sont pas décrit)

Articles

  • Introduction aux Web Workers d’HTML5 : le multithreading version JavaScript (fr) : Un très bon article en français sur les workers.
  • Using web workers (en) : Guide sur le MDN sur comment utilisé les workers.
  • HTML5 Rocks - The Basics of Web Workers (en) : Guide d'utilisation des workers.
  • Web Workers rise up! (en) : un article faisant un tour rapide des worker.
