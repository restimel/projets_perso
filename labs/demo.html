Flamme CSS

Une petite animation réalisée en CSS et JavaScript.
Le but était de voir ce qu'il est possible de faire avec la propriété box-shadow.
L'allumette s'éteint avec une force de vent de 10

Reconnaissance vectoriel de caractère

Une petite petite reconnaissance de caractères vectoriel. Il s'appuit sur les caractères PALM.
Il pourrait encore être amélioré afin de trouver un caractère avoisinant lorsque le mouvement n'a pas été reconnu.
Ce script utilise canvas pour dessinner les mouvements effectués.

Performance Scrolling

Quand une page possède beaucoup d'éléments DOM, elle peut très fortement ralentir. Ce code permet de créer un objet gérant un élement de scroll. Les éléments HTML sont créés et détruits au fur et à mesure de ce qui est nécessaire pour l'affichage.

Jeux

Tangram

Le Tangram est un jeu où le but est de reproduire une figure à l'aide de 7 pièces géométriques appelées Tans.
Canvas est utilisé pour réaliser ce jeu.
Le script détecte lorsque le joueur arrive à reproduire la figure. Plusieurs figures à reproduire sont possibles.
