Outils en ligne

Je vous présente ici une petite collection d'outils que j'ai développés en JavaScript. Ces outils sont disponibles directement en ligne et peuvent, pour la plupart, être déplacés facilement dans un environnement différent.
Ces outils ont été développés pour des besoins spécifiques et ont souvent été enrichis de fonctionnalités dépendant de l'usage. Ils restent donc très sobres et peuvent être difficiles d'accès.

Manipulation de Textes

Remplacement de texte

Permet d'effectuer plusieurs remplacements de chaînes de caractères en même temps. Permet de remplacer des passages par des nombres auto-incrémentés ou par une liste de valeurs prédéfinies.
Cet outil dispose d'un système de sauvegarde et de rechargement afin de reproduire les mêmes traitements ultérieurement.
Une vérification syntaxique des expressions régulières permet d'indiquer une erreur éventuelle.
L'outil a été conçu pour être facilement transportable : tout est intégré en une seule page et peut fonctionner en local.
Accéder à l'outil (en)

Caractères et Symboles UTF-8

Permet d'afficher les caractères à partir de leur code UTF-8, ce qui permet d'aider à trouver plus facilement un symbole désiré. Cet outil permet aussi de connaître le code UTF-8 d'un ou plusieurs caractères.
Les entités HTML (&amp; pour afficher &) sont affichées à côté des caractères.
L'outil a été conçu pour être facilement transportable : tout est intégré en une seule page et peut fonctionner en local.
Accéder à l'outil (fr)

Programmation Web

JS Color

Cet outil permet d'obtenir une coloration syntaxique d'un code JavaScript. Le résultat est sous forme de balises HTML qu'il suffit de copier/coller.
Cette fonction JavaScript pourrait être utilisée dans un site pour réaliser une coloration syntaxique à la volée.
Accéder à l'outil (en)

JS Performance

Cet outil permet de comparer la performance de différentes méthodes JavaScript afin de mieux choisir entre elles. Des modules ou de nouveaux tests peuvent être ajoutés régulièrement.
Accéder à l'outil (en)

CSS Box-shadow

Permet de définir les ombres à appliquer à un objet en CSS en manipulant facilement les paramètres à l'aide de curseurs.

Outils d'analyse

Analyse de JSON

Cet outil permet d'analyser un code JSON, de rechercher des éléments à l'intérieur de l'objet et de le comparer à un autre code JSON.
Accéder à l'outil (en)

Comparateur de dates

Cet outil permet de comparer des dates afin de connaître le temps écoulé entre ces deux dates. Cela est particulièrement utile pour des dates qui sont dans des formats différents, par exemple lorsqu'elles viennent de fichiers de log différents.
Il est capable de reconnaître des dates dans de nombreux formats.
