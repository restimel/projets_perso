InsideWeCube : résolveur de casse-tête 'insideZeCube'

Cette application permet de résoudre les cubes casse-tête 'InsideZeCube'. Il est également possible de construire ses propres cubes à l'aide de ceux que l'on possède.
Ce projet utilise les technologies HTML5: web-workers et indexDB, et également CSS3: tranform (rotateZ).
L'ensemble du code source est disponible sur Github. La version disponible ici est une version de développement et n'est pas forcément stable. Pour naviguer sur une version plus stable utilisez celle disponible en ligne sur Github.

Jugos : un éditeur de partie de Go

Cette application permet de lire et d'éditer des parties aux format SGF. Il est également possible de compter les points en fin de partie ou de lancer des quizz à partir d'un fichier (le but étant de retrouver le coup joué).
Le goban est affiché à l'aide du SVG.
Ce projet figure parmi mes premiers réalisés en JavaScript (en 2007). Et bien qu'incomplet il est tout à fait fonctionnel. Consulter les fonctionnalités SGF supportées.
