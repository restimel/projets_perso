Explication i++ retourne la valeur de i puis incrémente i.
Dans une boucle do … while, le code à l'intérieur de la boucle est exécuté avant de tester la condition. Cela signifie que ce code est au moins exécuté une fois.

Au début, le code de la boucle est exécuté (donc j vaut maintenant 1). On vérifie que la valeur de i (qui vaut 0) est inférieure à 1. Puis i est incrémentée.
Le code de la boucle est à nouveau exécuté (donc j vaut maintenant 2). On vérife que la valeur de i (qui vaut 1) est inférieure à 1. Ce qui n'est plus le cas, donc on arrête la boucle. Mais i est quand même incrémentée et vaut finalement 2.
